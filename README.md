## ⬡ Research

I develop **conformal prediction frameworks** that provide valid uncertainty guarantees for ML classifiers operating under distributional shift — motivated by physical systems where training and deployment conditions fundamentally differ.

**Current thesis direction:** Distribution-free false alarm rate bounds for neural network-based gravitational wave detection pipelines, addressing the gap between synthetic training data and real LIGO detector noise.

**Prior work:** Identified the *Evaluation Paradox* in subsurface soil moisture retrieval — demonstrating that standard random data splits mask temporal dependencies, producing spurious model selection and invalid uncertainty estimates. Showed conformal prediction maintains coverage where standard methods fail.
